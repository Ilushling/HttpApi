# Http api
Http api.

1) [Usage](#usage):
    1) [Import types](#import-types);
    2) [Create](#create).

## Usage
### Import types
```js
/**
 * @typedef {import('httpapix').IHttpApi} IHttpApi
 * @typedef {import('httpapix').IDecoratorHttpApi} IDecoratorHttpApi
 */
```

#### Create
```js
import {
  HttpApi
} from 'httpapix';

const httpApi = new HttpApi();
```
